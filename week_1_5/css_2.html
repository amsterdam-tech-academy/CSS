Bron:Values and units in CSS

Breedte: px (absoluut) vw (relatief) em (relatief)

I am 200px wide.

Width set in pixels. As an absolute unit, this width will remain the same no matter what else changes.
I am 100vw wide.

Width set in vw (viewport width) units. This value is relative to the viewport width, and so 60vw is 60 percent of the width of the viewport. If you change the width of your browser window, the size of the box should change.
I am 10em wide.

Width em units. These are relative to the font size. I've set a font size of 1em on the containing
, which has a class of .wrapper. Change this value to 1.5em and you will see that the font size of all the elements increases, but only the last item will get wider, as its width is relative to that font size.
